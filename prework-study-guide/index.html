HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the framework for the webpage

CSS

  • CSS is the visual aspect of the webpage
  • By utilizing commas you can practice DRY code by simplifying and eliminating repitition
  • A margin indicates how much space we want around teh outside of an element
  • A padding indicates how much space we want around the content inside and element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: returns you to the main branch

JavaScript

  • A variable, represented by 'var', is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script; in JavaScript this is read from top to bottom
  • You can group multiple items into a var by utilizing an array.
  • Remember that in order for a function to work you must call it. Function names are also followed by a set of parentheses. EX: function listTopics()
